Movie Planet Studio is an AI-powered creative desktop application for Windows. Text, image, video, and audio generation — unified in one workspace, powered by Google Gemini.

First Launch

1
Launch the application

The backend server starts automatically — wait for the main window to appear.

2
Enter your API Key

Click the Settings icon (top-right of any module). Paste your Google Gemini API key and save. The status badge turns green when accepted.

3
Create a Project

From the Dashboard, click New Project, enter a name, and press Create.

4
Choose a Module

Use the left navigation rail to switch between Text, Image, Video, and Audio.

02

Projects

Projects save your entire workspace — all four modules, uploaded references, and generated outputs — into a single .mps file.

Creating a Project

Go to Dashboard → New Project and enter a name, or press Ctrl+N from anywhere. Projects are saved to Documents\Movie Planet Studio\Projects\.

Opening a Project

MethodHow
DashboardClick a project card in the project list
File menuCtrl+O — opens a native file picker for .mps files
Drag & DropDrag a .mps file onto the Dashboard

Saving a Project

ActionShortcutBehaviour
SaveCtrl+SSaves to the current file
Save AsCtrl+Shift+SSaves to a new location
Auto-Save—Silently saves every 60 seconds when there are unsaved changes (dirty-flag dot in title bar)

Project File Format (.mps)

A .mps file is a ZIP archive containing project_data.json (all module settings) and an assets/ folder (embedded reference images, videos, and audio). Maximum project size is 500 MB.

💡
To rename or delete a project, hover its card on the Dashboard and use the ⋯ menu.

💡
Tip for Text to Video: Be specific about camera movement (e.g., "slow dolly push-in"), lighting, subject action, and environment to get the best results.

Image to Video — Sub-modes

Key Frame Mode — Upload a start image, end image, or both. The AI animates the transition between them.

Reference Mode — Upload up to 7 reference frames (ref_1 through ref_7). The AI generates motion inspired by the visual content of all frames.

The Gallery stores all outputs generated within the current session. Open it from the Gallery button in any module's output header bar. Filter by type: All · Image · Video · Audio · Scene.

ActionHow

09

Presets

Presets save and recall the full configuration of a module tab — inputs, aspect ratio, resolution, and advanced settings. The Presets panel has two tabs: System (built-in) and My Presets (your saved ones).

Saving a Preset

1
Configure the module

Set everything to your desired state.

2
Open Presets panel

Click Presets in the output header bar, then Save Current as Preset.

3
Name and save

Enter a name and optional description, then click Save.

User presets are stored as individual .json files in Documents\Movie Planet Studio\Presets\{Module}\ — easy to back up, share, or transfer between installations.

Settings & API Keys

Access Settings via the gear icon in any module's output header bar.

API Key Setup

MP Studio uses the Google Gemini API for all AI generation.

1
Get your key

Go to Google AI Studio to create a free API key.

2
Paste it in Settings

Open Settings → Gemini API Key and paste your key.

3
Save

The System Status Badge turns green when the key is accepted.

🔒
Your API key is encrypted with AES-256-CBC and stored locally at %APPDATA%\Movie Planet Studio\Settings\config.json. The key never leaves your machine unencrypted.

Troubleshooting

The app shows a server error on startup ▶
The internal Express server failed to start. Try:
  1. Relaunch the application.
  2. Check if another process is using ports 3000–3099.
  3. Check logs at %APPDATA%\Movie Planet Studio\logs\.
Generation fails with "API key not set" ▶
Open Settings and re-enter your Gemini API key. Ensure the key has the required API permissions enabled in Google AI Studio.
Generation fails with "Rate limit exceeded" ▶
Your Gemini API key has hit its request quota. Wait a moment and try again, or check your quota in Google AI Studio.
A project won't open ▶
Ensure the .mps file is not corrupted (it is a ZIP archive — verify with any ZIP tool). Check the file is under 500 MB. The file may reference a path that no longer exists; assets are embedded, so this is rare.
Generated images are not saving ▶
Check that your Gallery Location folder exists and the app has write access. If using the default session cache, ensure %LOCALAPPDATA% is not restricted.
Video jobs stay "Pending" indefinitely ▶
Verify your API key supports the video model you selected. Check the Render Queue for an error message on the job card. Some video models (e.g., Veo) have longer generation times — allow up to 5 minutes.
Audio playback is silent ▶
Check your system volume and default audio output device. Try downloading the file and playing it in a media player to rule out an in-app audio issue.
The app feels sluggish after long sessions ▶
Open the Render Queue and remove completed jobs. Restart the application to clear the session cache from memory.
